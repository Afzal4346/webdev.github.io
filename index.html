Basic Structure of HTML Document

It consists of following elements:-

  1. Preamble. <!Doctype html>
  2. <html lang = "en-US" > ......... </html>. Root Element that wraps the content of the whole page alongwith lang
    attribute setting Primary document language.
  3. <head> ........ </head>. Container for all the stuff to be included in the page but not displayed for viewers. Like Keywords, description to appear in search results, CSS to style the content, character set declaration etc.
  4. <body> ....... </body>, Container for all the stuff to be displayed for viewers. Images, text, videos, audio etc.
